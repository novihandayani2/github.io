## ğŸš€ Setup & Deployment

### Step 1: Create GitHub Repository

1. Login ke [GitHub](https://github.com)
2. Klik tombol **New Repository**
3. Nama repository: `novihandayani.github.io`
4. Pastikan repository **Public**
5. Klik **Create repository**

### Step 2: Upload Files

#### Option A: Upload via GitHub Web Interface

1. Di halaman repository baru, klik **uploading an existing file**
2. Drag & drop semua file dari folder `novihandayani.github.io/`
3. Klik **Commit changes**

#### Option B: Upload via Git Command Line

```bash
# Clone repository (kosong)
git clone https://github.com/username/novihandayani.github.io.git
cd novihandayani.github.io

# Copy semua file ke folder ini
# cp -r /path/to/novihandayani.github.io/* .

# Add, commit, push
git add .
git commit -m "Initial portfolio website"
git push origin main
```

### Step 3: Upload Profile Photo

**PENTING:** Upload foto profil ke folder `assets/`

1. Rename foto menjadi: `foto-profil.jpg`
2. Upload ke folder: `/assets/`
3. Pastikan ukuran file **< 500KB** untuk performa optimal
4. Format yang diterima: `.jpg`, `.jpeg`, `.png`

### Step 4: Akses Website

1. Buka browser
2. Akses: `https://novihandayani.github.io`
3. Website akan live dalam 1-5 menit
